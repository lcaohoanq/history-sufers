BOXY RUN

Temple Run Inspired Racing

Original by lcaohoanq

Enhanced with Multiplayer Racing & Modern UI

How to Play

Controls

↑ Arrow Key Jump over obstacles
← Arrow Key Move to left lane
→ Arrow Key Move to right lane
P Pause game (single player)

Single Player

Avoid trees and survive as long as possible! Score increases as you run further. Challenge yourself to beat your high score and unlock new ranks!

Multiplayer

  • Create or join a room with friends
  • Race against up to 6 players!
  • Real-time leaderboard
  • Highest score wins!
